# Data Odyssey: Tableau Dashboard 

## Project Overview
This project was created as part of a hands-on workshop titled "Data Odyssey: From Excel to Tableau", where I served as a resource person. The goal was to demonstrate how raw data can be transformed into meaningful insights using visualization techniques. I carried out a 3 hour hands-on workshop for college students explaining them how to deal with unclean datasets in excel, doing deep cleaning of the dataset and then using it for data visualisation.
